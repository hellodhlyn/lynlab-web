# LYnLab Web

> Nuxt.js application for my personal blog

## Development

``` bash
# install dependencies
$ yarn install

# serve with hot reload at 127.0.0.1:8080
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
